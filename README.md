## Pre-requisite
1. MySQL installed
2. Go 1.16.0+ installed
3. Make sure your environment supports to develop apps in Go

## Configuration
- You can find the configuration at *.env* file

## How to action
1. Get all dependencies needed  
```bash
go mod tidy
```

2. Go to `.\cmd\app` (relatively to project folder)

4. Run the app
```bash
go run .
```
